# Angular project

An angular project to store links built using Angular, HTML, and SCSS.

## Table of Contents

- [Overview](#overview)
- [Tech Stack](#tech-stack)
- [Features](#features)
- [Requirements](#requirements)

## Overview

This project allows users to add, view, edit, and delete links. Each link is displayed as a Favicon. The theme is designed around a pokemon PC.

### Installation

```bash
git clone https://github.com/ijnebism/benjamin-frontend-display.git

cd benjamin-frontend-display

npm install

npm run start
```
